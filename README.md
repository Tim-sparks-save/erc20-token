# erc20-token
## ERC20 Token Smart Contract

A standard ERC20 token smart contract implemented in Solidity using the secure OpenZeppelin library.

## Features
* Custom Token Name: Vortiq Coin
* Custom Token Symbol: VRTQ
* Total Supply: 1,000,000 VRTQ (minted to the deployer wallet)
* Standard Compliant: Full compatibility with ERC20 rules for transfers and balances.

## Tech Stack
* Language: Solidity
* Library: OpenZeppelin Contracts
* Development Tool: Remix IDE
